# Python `match` `case` サンプル集

(Japanese) Python 3.10 で導入された `match` `case` 構文のサンプル集です。

## 必須

- Docker
- Docker Compose

## 使い方

リポジトリをチェックアウトします。

ビルドします。

```bash
cd python-match-case-examples-ja/
docker-compose build
```

Pytest でサンプルを実行します。

```bash
docker-compose run --rm python
```
